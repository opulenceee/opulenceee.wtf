Eleve Discord Bot

A specialized Discord bot designed for job scheduling and management. It allows administrators to create, edit, and manage job listings while members can easily respond to job availability through reactions.

  • Job creation and management with date, time, and location tracking
  • Interactive response system using reactions (Accept ✅, Decline ❌, Tentative ❓)
  • Paginated job listing view with easy navigation
  • Administrative controls for job editing and deletion

LS-RP Discord Bot

This bot tracks logged-in players, shows online admins and testers, and allows status checks via Discord commands. You can also monitor your faction thread with the bot.

  • Real-time player tracking and server status monitoring
  • Online admin and tester visibility
  • Faction thread monitoring system
  • Simple Discord command interface for status checks

NR-RP Ticket Bot

A comprehensive ticket management system for New Rotterdam Roleplay. This bot handles user support requests through various categories, manages staff responses, and maintains detailed ticket records.

  • Four support categories with role-based access (General, Premium, Account Recovery, Management)
  • Staff ticket claiming system with notification management
  • Automatic HTML transcript generation and archiving
  • Advanced user and staff interfaces with embeds and modal forms

LS-NIGHTS Role Bot

A streamlined role management system for Discord servers. This bot enables users to self-assign roles through an intuitive emoji reaction system, making server role management effortless and user-friendly.

  • Self-service role assignment system with emoji reactions (VIP, Auction Winners, Member, and More Pings roles)
  • Automated role verification and error handling system
  • Clean embedded message interface for role selection
  • Real-time role updates with add/remove functionality through reactions
